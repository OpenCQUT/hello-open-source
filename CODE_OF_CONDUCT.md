# 社区行为准则

Hello Open Source 面向新手。我们优先维护一个安全、耐心、可学习的协作环境。

## 鼓励的行为

- 对新手问题保持耐心；
- 指出问题时给出具体修改建议；
- 允许别人从小任务开始；
- 尊重不同技术背景、学习速度和表达方式；
- 主动帮助内容变得更清晰。

## 不接受的行为

- 人身攻击、嘲讽、羞辱；
- 因经验不足而贬低贡献者；
- 发布歧视性、骚扰性或恶意内容；
- 未经授权公开他人隐私信息；
- 利用本项目推广低质量或误导性商业课程。

## 处理方式

维护者可以对违反准则的内容进行编辑、隐藏、关闭 Issue、关闭 PR，或限制相关账号继续参与。

如果你发现问题，请通过 Issue 或仓库维护者提供的联系方式反馈。
