# 项目路线图

## M1：网站可用

- Docusaurus 框架完成；
- 首页、资源页、路线图、贡献页完成；
- GitHub Pages 自动部署；
- README、贡献指南、行为准则完成。

## M2：内容成型

- 10 篇核心文档；
- 8 个首批资源导读；
- 30 天路线；
- 第一次 PR 教程；
- 新手任务池。

## M3：社区启动

- 维护 10 个 good first issue；
- 组织第一次 PR 工作坊；
- 收集参与者反馈；
- 发布活动复盘。

## M4：长期维护

- 建立资源审核机制；
- 形成学习小组；
- 引入贡献者展示；
- 增加搜索、标签和数据化资源管理。
