# 内容规范

## 基本原则

- 面向新手；
- 具体可执行；
- 少堆链接；
- 每个资源都要有导读；
- 每个教程都要有下一步。

## 推荐结构

```md
# 标题

## 适合谁

## 学习目标

## 操作步骤

## 常见问题

## 下一步
```

## 不推荐写法

```md
这里有很多资料，大家自己看。
```

## 推荐写法

```md
如果你刚开始学习 Git，先读前两章并完成一次 commit。暂时不需要学习 rebase。
```

## 外部内容处理

- 不复制大段原文；
- 用自己的话写导读；
- 保留原始链接；
- 标注适合人群和使用方式。
