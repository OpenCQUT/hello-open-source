# 贡献总览

本项目欢迎新手贡献。你不需要一开始就写代码。

## 推荐贡献类型

| 类型 | 适合人群 |
|---|---|
| 修正文档 | 第一次参与开源的人。 |
| 补充资源导读 | 正在学习公开课的人。 |
| 改进教程步骤 | 走通过某个流程的人。 |
| 优化页面样式 | 会一点 HTML / CSS / React 的人。 |
| 组织活动材料 | 社区组织者。 |

## 第一个贡献建议

从以下任务中选择一个：

- 修正一个错别字；
- 给资源补充适合人群；
- 给某个教程增加步骤说明；
- 写一段自己的学习复盘。

## 贡献闭环

```mermaid
flowchart LR
  A[选择任务] --> B[Fork]
  B --> C[新建分支]
  C --> D[修改]
  D --> E[提交 PR]
  E --> F[Review]
  F --> G[合并或复盘]
```
