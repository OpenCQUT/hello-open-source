# Markdown 写作指南

本站大部分内容用 Markdown 编写。

## 常用语法

````md
# 一级标题

## 二级标题

- 列表项
- 列表项

[链接文字](https://example.com)

```bash
npm run start
```
````

## 写作建议

- 标题要具体；
- 段落不要太长；
- 表格用于对比；
- 代码块用于命令；
- 不要复制外部大段内容。

## 页面结构模板

```md
# 主题名称

## 适合谁

## 学习目标

## 操作步骤

## 常见问题

## 下一步
```
