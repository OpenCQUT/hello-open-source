# Hello Open Source 是什么

Hello Open Source 是一个面向编程与开源新人的中文学习网站。

它的目标不是收集尽可能多的链接，而是回答三个问题：

1. 我现在处于哪个阶段？
2. 我下一步应该学什么？
3. 我怎样完成第一次真实的开源贡献？

## 项目定位

这个项目同时承担两个角色：

| 角色 | 说明 |
|---|---|
| 学习站点 | 提供开源基础、GitHub 使用、公开课导读、学习路线和实践任务。 |
| 练习仓库 | 让新手通过修正文档、补充资源、提交 PR 来完成第一次贡献。 |

## 不做什么

本站暂时不做这些事情：

- 不做无筛选的链接合集；
- 不把难度跨度很大的资源混在一起；
- 不推荐没有明确适合人群的课程；
- 不要求新手一开始就贡献复杂代码；
- 不把“完成 PR 截图”当作唯一目标。

## 推荐学习方式

```mermaid
flowchart LR
  A[认识开源] --> B[掌握工具链]
  B --> C[阅读仓库]
  C --> D[完成小贡献]
  D --> E[参与项目实践]
  E --> F[维护社区]
```

先完成一件小事，再进入更系统的学习。
