# Web 基础

本站是一个静态网站。参与样式和页面贡献时，需要了解基本 Web 知识。

## 三个核心部分

| 技术 | 作用 |
|---|---|
| HTML | 页面结构。 |
| CSS | 页面样式。 |
| JavaScript | 页面交互。 |

## 本项目中的对应位置

| 文件 | 作用 |
|---|---|
| `src/pages/index.jsx` | 首页结构。 |
| `src/pages/index.module.css` | 首页样式。 |
| `src/css/custom.css` | 全局主题样式。 |
| `docs/**/*.md` | 文档内容。 |

## 新手可做的页面任务

- 调整卡片间距；
- 修改资源卡片文案；
- 增加一个页面区块；
- 优化移动端布局；
- 改善颜色对比度。

## 注意

科幻风不等于低可读性。正文必须清晰，装饰效果不能影响阅读。
