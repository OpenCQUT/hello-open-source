# 命令行与工具链

命令行不是为了炫技，而是为了更稳定地操作项目。

## 新手先掌握这些命令

```bash
pwd       # 当前目录
ls        # 查看文件
cd        # 切换目录
mkdir     # 创建文件夹
cp        # 复制文件
mv        # 移动或重命名
rm        # 删除文件
cat       # 查看文件内容
```

## 项目开发常用命令

```bash
npm install
npm run start
npm run build
```

## 推荐工具

| 工具 | 用途 |
|---|---|
| VS Code | 编辑器。 |
| Git | 版本控制。 |
| Node.js | 运行 Docusaurus。 |
| Markdown 插件 | 预览文档。 |
| 浏览器开发者工具 | 调试样式和页面。 |

## 验收任务

在本地启动本站，并修改一个 Markdown 页面：

```bash
npm install
npm run start
```

然后访问本地页面，确认你的修改能显示出来。
