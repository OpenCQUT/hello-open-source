# 计算机基础

计算机基础用于帮助你理解“为什么这样写代码”。

## 入门阶段先学什么

| 主题 | 先掌握到什么程度 |
|---|---|
| 数据结构 | 数组、链表、栈、队列、哈希表的用途。 |
| 算法 | 排序、查找、递归、复杂度的基本概念。 |
| 操作系统 | 进程、文件、权限、命令行的基本概念。 |
| 网络 | URL、HTTP、DNS、请求与响应。 |
| 数据库 | 表、字段、SQL 查询、索引的基本作用。 |

## 不建议一开始死磕什么

- 操作系统源码；
- 编译器完整实现；
- 大型分布式系统；
- 高难度算法竞赛题。

这些内容很有价值，但不适合作为开源入门的第一步。

## 与开源贡献的关系

计算机基础能帮你读懂项目，但你不需要等基础全部学完才参与开源。文档、资源和简单页面改进都可以同步进行。
