# 编程基础

编程基础不是背语法，而是理解如何把问题拆成可以执行的步骤。

## 学习目标

学完这一阶段，你应该能：

- 写出变量、条件、循环、函数；
- 看懂简单错误信息；
- 使用数组、对象、字典等基础数据结构；
- 写一个 50–200 行的小程序；
- 把代码和说明提交到 GitHub。

## 推荐语言

| 语言 | 适合情况 |
|---|---|
| Python | 入门友好，适合算法、脚本和数据处理。 |
| JavaScript | 适合网页开发，也能直接用于本站页面。 |
| C | 适合理解底层，但入门阻力更高。 |

## 推荐任务

- 写一个命令行待办事项工具；
- 写一个课程资源整理脚本；
- 写一个网页卡片组件；
- 给本站资源页面补充一条资源数据。

## 验收标准

你不需要精通语言，但应该能独立完成：

```txt
输入 → 处理 → 输出
```

这种最基本的程序结构。
