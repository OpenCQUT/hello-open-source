# 新手任务池

以下任务适合设置为 `good first issue`。

## 文档任务

- 给 CS50 页面补充中文学习建议；
- 给 Missing Semester 页面补充实践任务；
- 修正 README 中不清楚的安装步骤；
- 给第一次 PR 指南补充常见错误。

## Bug 修复任务

- 修复失效链接、错误路径或跳转到错误页面的按钮；
- 修复移动端文字溢出、按钮对比度不足、侧边栏显示异常；
- 修复构建警告、Markdown 表格渲染问题或错误的页面标题；
- 给一个已复现的问题补充最小修复和验证说明。

## 小功能任务

- 给资源页增加一个更清晰的筛选或排序入口；
- 给贡献入口增加“适合第一次做什么”的提示；
- 给文档页面补充 FAQ、检查清单或示例区块；
- 给某类资源增加字段，例如难度、时长、前置知识。

## 缺失内容任务

- 新增一个仓库里缺少但对新手重要的页面；
- 补齐现有页面里的空白章节；
- 把一次真实踩坑整理成问题说明和解决步骤；
- 提出一个内容缺口，并在 Issue 中写清楚为什么需要它。

## 资源任务

- 增加一个中文 Git 教程导读；
- 检查资源链接是否失效；
- 给每个资源标注适合阶段；
- 补充资源筛选标准。

## 设计任务

- 优化移动端资源卡片；
- 给首页增加一个活动入口；
- 改善文档表格样式；
- 制作一张开源流程图。
