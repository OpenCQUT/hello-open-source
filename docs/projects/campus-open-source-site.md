# 校园开源网站项目

这是一个适合作为社团或技术社区共建的项目。

## 目标

建设一个校园开源入口，展示：

- 开源入门知识；
- 学习路线；
- 公开课导读；
- 校园活动；
- 贡献者故事；
- 新手任务池。

## 分工建议

| 角色 | 任务 |
|---|---|
| 内容组 | 写文档、整理资源。 |
| 设计组 | 维护视觉风格和页面体验。 |
| 技术组 | 维护 Docusaurus、部署和页面组件。 |
| 运营组 | 组织活动、维护 Issue、收集反馈。 |

## 第一个里程碑

- 网站可访问；
- 至少 10 篇核心文档；
- 至少 8 个资源导读；
- 至少 5 个 good first issue；
- 至少一次线下或线上 Workshop。
