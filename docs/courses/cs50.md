# CS50 导读

CS50 是非常适合作为计算机科学入门的公开课，但它不是“轻松课”。

## 适合谁

- 希望系统了解计算机科学的新手；
- 能接受英文材料或愿意查词的学习者；
- 愿意花数周时间做作业的人。

## 不适合谁

- 只想快速学会某个框架；
- 完全不愿意写作业；
- 只想看视频不实践。

## 建议学习方式

1. 先看前两周，判断难度；
2. 每周至少完成一个 Problem Set；
3. 遇到困难先记录，再搜索，再提问；
4. 学完一个主题后写中文笔记；
5. 把笔记或资源导读贡献到本站。

## 与开源贡献的连接

完成 CS50 前几周后，你可以尝试：

- 给本站补充 CS50 学习笔记；
- 写一篇“零基础学 CS50 的坑点”；
- 维护 CS50 相关中文资源链接。
