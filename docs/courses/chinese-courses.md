# 中文资源

中文资源对新手很重要，但需要筛选质量。

## 收录标准

中文资源应尽量满足：

- 内容免费或低成本；
- 结构清晰；
- 不以营销为主要目的；
- 有明确适合人群；
- 能帮助完成具体任务。

## 推荐补充方向

| 方向 | 示例 |
|---|---|
| Git 入门 | 中文 Git 教程、Pro Git 中文版。 |
| Markdown | 文档写作入门。 |
| Web 基础 | HTML / CSS / JavaScript 中文教程。 |
| 开源协作 | 中文开源指南、社区文章。 |
| 计算机基础 | 中文公开课、课程笔记。 |

## 提交资源时请写导读

不要只提交链接。请补充：

```md
## 适合谁

## 学前准备

## 建议学习方式

## 学完可以做什么
```
