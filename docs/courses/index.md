# 公开课与资源总览

本站推荐资源遵循一个原则：少而清晰。

每个资源页面都应该回答：

- 这个公开课或资源到底讲了什么；
- 它的目标是什么，不解决什么问题；
- 每一类任务会带学习者做什么；
- 推荐时长、难度和前置要求是什么；
- 学完后如何连接到一次真实开源贡献。

## 导读模板

新增公开课或资源时，不要只写“适合新手”“质量很高”。请按下面结构补充：

```md
## 这是什么

## 它的目标

## 它讲什么

## 任务会带你做什么

## 推荐时长和难度

## 学完以后做什么
```

## 首批资源分类

| 分类 | 示例 |
|---|---|
| 编程入门 | CS50、Python / JavaScript 入门课程。 |
| 工具链 | Missing Semester、Pro Git、命令行教程。 |
| 计算机基础 | MIT OCW、OSSU、算法与系统课程。 |
| 开源协作 | GitHub Skills、开源软件指南。 |
| 中文资源 | 中文课程、中文文档、社区教程。 |

## 维护要求

新增资源前，请先检查是否已经存在同类资源。能用一个资源解决的问题，不要列十个相似资源。
