# Missing Semester 导读

Missing Semester 适合补齐工具链：Shell、Git、编辑器、调试、脚本和自动化。

## 为什么适合开源入门

开源贡献经常需要：

- 在命令行中运行项目；
- 使用 Git 管理分支；
- 阅读错误信息；
- 调整编辑器和环境；
- 写脚本自动化重复操作。

这些能力传统编程课不一定系统讲，但参与开源时非常实用。

## 推荐学习顺序

1. Shell；
2. Editors；
3. Version Control；
4. Debugging；
5. Metaprogramming。

## 验收任务

学完 Git 相关内容后，在本站完成：

```txt
Fork → Branch → Commit → Push → Pull Request
```

这条完整链路。
