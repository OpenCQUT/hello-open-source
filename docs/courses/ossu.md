# OSSU 导读

OSSU Computer Science 是一条较完整的计算机科学自学路线。

## 适合谁

- 已经具备基本编程能力；
- 想长期系统补齐 CS 基础；
- 能接受英文课程；
- 有 1 年以上持续学习计划。

## 新手注意

OSSU 很完整，但完整也意味着压力较大。不要把它当作开源入门第一步。

本站建议：

1. 先完成 30 天开源入门计划；
2. 再学习 Missing Semester；
3. 然后根据兴趣选择 OSSU 中的一小段课程。
