# 资源审核标准

资源库的质量取决于筛选标准。

## 基本标准

一个资源被收录前，至少需要说明：

- 名称；
- 链接；
- 类型；
- 语言；
- 适合阶段；
- 预计学习周期；
- 前置知识；
- 推荐理由；
- 使用建议。

## 不建议收录

- 无法确认来源的资源；
- 明显营销导向的课程；
- 只给出链接，没有导读；
- 内容陈旧且无人维护；
- 与已有资源高度重复。

## Review 清单

维护者可以按以下问题审核：

1. 这个资源是否真的适合新手？
2. 它解决了路线中的哪个问题？
3. 有没有更权威或更清晰的替代？
4. 是否存在版权或转载风险？
5. 是否给出了可执行学习建议？
