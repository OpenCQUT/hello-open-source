# 活动模板

## 第一次 PR 工作坊

### 目标

让参与者在活动中完成一次文档类 PR。

### 时长

90–120 分钟。

### 流程

| 时间 | 内容 |
|---|---|
| 0–15 分钟 | 介绍开源和本站结构。 |
| 15–35 分钟 | 配置 Git、Node.js、编辑器。 |
| 35–55 分钟 | Fork、Clone、创建分支。 |
| 55–85 分钟 | 修改文档并提交 PR。 |
| 85–110 分钟 | Review、修改和合并。 |
| 110–120 分钟 | 复盘和下一步。 |

## 准备清单

- 提前准备 10 个 good first issue；
- 提前确认环境安装说明；
- 准备投影演示；
- 安排维护者现场 Review；
- 活动后收集反馈。
